Política de Privacidade — Binderdex

Última atualização: 28 de junho de 2026

A Binderdex ("a app") é um registo pessoal de coleções de cartas Pokémon TCG, com uma comunidade opcional para trocar/vender cartas. A app é local-first: funciona sem conta e a tua coleção fica no teu dispositivo. Só sincroniza para a nuvem e usa funções sociais se escolheres iniciar sessão. Esta política explica que dados são tratados e porquê.

Responsável

Desenvolvida por hivecode. Contacto: hivecode.comercial@gmail.com.

A app não recolhe localização, contactos, nem fotografias do teu dispositivo. As imagens das cartas vêm de uma base de dados pública de cartas (pokemontcg.io), não são carregadas por ti.

Onde ficam guardados

Localmente no teu dispositivo (sempre) e, com sessão iniciada, no Google Firebase (Authentication, Cloud Firestore, Cloud Messaging, Cloud Functions), que atua como subcontratante em nome da hivecode. A transmissão é encriptada (HTTPS/TLS).

Visibilidade para outros utilizadores

Ao usares a comunidade, alguns dados são, por natureza, visíveis a outros utilizadores autenticados: o teu nome/avatar, os teus anúncios e as mensagens que envias (visíveis ao destinatário). Não partilhamos os teus dados com terceiros para fins próprios deles (publicidade, etc.).

Moderação

Para manter a comunidade segura, o administrador pode ver denúncias e, para as diagnosticar, a conversa entre as pessoas envolvidas. Contas que violem as regras podem ser avisadas ou suspensas; uma conta suspensa pode apelar por escrito.

Os teus direitos / eliminar dados

Podes eliminar a conta em Perfil → Eliminar conta, o que remove a tua coleção da nuvem. Para apagares também anúncios/mensagens/outros dados residuais, contacta hivecode.comercial@gmail.com e tratamos do pedido.

Crianças

A app não se destina a crianças com idade inferior à exigida para criar conta Google na tua região.
